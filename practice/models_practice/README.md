# Odoo Models - Practice Exercises

This directory contains practice exercises for writing Odoo models.

## Structure
- `exercises/` - Practice problems
- `solutions/` - Solutions (try yourself first!)
- `my_solutions/` - Your solutions

## How to Practice
1. Read the exercise description
2. Try to write the model yourself in `my_solutions/`
3. Compare with solution in `solutions/`
4. Ask questions if stuck!

## Exercises
1. **exercise_01_library_book.md** - Simple model with basic fields
2. **exercise_02_student_enrollment.md** - Model with relationships
3. **exercise_03_task_management.md** - Model with computed fields and constraints
4. **exercise_04_ecommerce_order.md** - Complex model with inheritance
5. **exercise_05_attendance_system.md** - Complete system with wizards

Happy practicing! 🚀
